# Hello World em Python

Este é um exemplo simples de execução de um script em Python, criado como parte dos estudos iniciais do projeto de SRE/DevOps.

## 📄 Arquivo

`hello.py`

```python
print("hello world")
```

---

## 🚀 Como executar

### Pré-requisitos

* Ter o Python 3 instalado

Verifique com:

```bash
python3 --version
```

---

### Execução

No terminal, dentro da pasta do projeto:

```bash
python3 hello.py
```

Saída esperada:

```bash
hello world
```

---

## ⚠️ Observação

Em distribuições Linux mais recentes (como Ubuntu), o comando `python` pode não estar disponível por padrão.
Nesse caso, utilize sempre `python3`.

Opcionalmente, é possível habilitar o comando `python` com:

```bash
sudo apt install python-is-python3
```
